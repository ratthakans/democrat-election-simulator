# Election Analytics Simulator

A high-performance, modern election simulation platform designed for the Democrat Party.

## Quick Start

1. **Install Node.js**: Download from [nodejs.org](https://nodejs.org/)
2. **Install Dependencies**:
   ```bash
   npm install
   ```
3. **Run Development Server**:
   ```bash
   npm run dev
   ```

## Key Technologies

- **Vite & React**: Fast development and rendering
- **Tailwind CSS**: Modern glassmorphism design system
- **Zustand**: Efficient state management with persistence
- **Framer Motion**: Premium micro-animations
- **Chart.js**: Dynamic data visualization

## Features

- National & District-level simulation
- Vote transfer modeling
- Scenario saving & management
- Monte Carlo statistical simulations
- Dark/Light mode support
